## [0.4.2] - 2026-07-05
운동 일지 신설 + 동적 가이드 + UI 정리.

### Added
- **📔 운동 일지** — 좌측 레일 하단 진입점. 월 캘린더에 활동 이모지 표시, 날짜 클릭 시 5개 활동 다중 체크 + 한 줄 메모 저장. 재시작 후에도 유지 확인
- **가이드 탭 동적 블록** — 오늘 날씨(비/강풍/자외선/폭염/미세먼지)에 맞춘 안내가 정적 가이드 위에 조건부 표시
- **애견 플랜 심화** — 신호 판정 이유(reason), 자제 시 대안(alternative), 다녀온 후 케어팁(careTip) 추가
- **활동별 위치 기억** — 걷기·등산·자전거 등 활동마다 마지막 위치를 따로 저장(즐겨찾기는 전역 공유 유지)

### Changed
- 좌측 카테고리 레일을 **큰 글자만**으로 정리(설명 제거), 활동 탭 아래 `activity-tagline` 완전 제거 → 탭 선택 즉시 오늘판단/준비/기록/가이드 노출
- 히어로 조건 칩이 모바일에서 2열 그리드로 가지런히 정렬
- 등산 데이파트는 새벽·아침·낮·**저녁** 4구간 유지로 재확정(비대칭 방지, 저녁도 다른 구간과 같은 기준 적용을 테스트로 고정)

### Removed
- 죽은 코드: `composeBriefing`/`Briefing`/`BRIEFING_VERDICT`, `findTomorrowMorningBest`, `shortReason`/`heroReason`, 미사용 import

### Invariant
- 러닝 골든마스터 통과. 테스트 36개(운동 일지 CRUD·활동별 독립·등산 4구간 회귀 포함)

## [0.4.1] - 2026-07-05
"상단 탭으로 점수만 바뀌는 구조" → **활동별 허브 앱**으로 구조 재설계.

### Added
- **활동 내부 탭** — 각 활동 안에 오늘 판단 / 준비 / 기록 / 가이드 세그먼트
  - 준비: 복장 상세 + 활동별 준비물·주의(`lib/activity-guide.ts`)
  - 가이드: 활동별 실전 팁 (걷기 오래걷기·애견 발바닥/노령견·러닝 인터벌·등산 하산/낙뢰·자전거 측풍/노면)
- **PC 좌측 카테고리 레일** (≥1200px) — 이모지+이름+한줄설명+선택 상태. 모바일은 상단 5탭 유지
- **활동별 기록 시스템** — 걷기·애견·러닝·등산·자전거 기록을 따로 저장(`running-alarm:activity-log:v1`), 활동별 연속일·주간/월간 목표(걷기 주5·애견 주7·러닝 주3·등산 월2·자전거 주2), 오늘 체크 버튼 활동별 문구
- 저장 안전 래퍼(SSR·손상 데이터·quota 방어), 정규화

### Changed
- 기록이 **재시작 후에도 유지** (localStorage), 기존 러닝 기록은 run으로 무손실 마이그레이션
- 공유·드로어·README에 등산 포함, 야외활동 확장 반영

### Removed
- 죽은 코드 `lib/tips.ts`, README의 `/api/tip` 설명

### Invariant
- 러닝 골든마스터 통과. 테스트 29개 (기록 마이그레이션·활동별 독립 스트릭·순서 포함)

## [0.4.0] - 2026-07-05
등산 카테고리 신설 + 전 카테고리 문구 심화 + 앱 전체 야외활동 확장.

### Added
- **⛰️ 등산 카테고리** — 목적지 산 중심의 등산 날씨 판단
  - 산 우선 검색: "북한산" 검색 시 산 결과 우선 (산·봉·악·오름·고개·령 + 카카오 카테고리 판정, 대학 오판 방지)
  - **하산 마감 카드**: 일몰 2시간 30분 전 하산 시작 안내 (내일 탭은 내일 일몰 기준 — 별도 저장)
  - **일출 산행 안내**(새벽), **안전 신호**(⛈️낙뢰·💨돌풍·🌧️미끄럼·❄️결빙·🌬️능선바람), **조망 예보**, 준비물 체크리스트(상황별 헤드랜턴·아이젠·우비)
  - **새벽 산행 시간대**: 추천 창 04시부터, 데이파트 새벽04-07/아침08-10/낮11-15/저녁16-19 4구간
  - 등산 점수 프로필(강수 0.28·바람 0.22 가중, 강풍캡 12m/s), 레이어링 복장·등산 준비물
- 날씨 데이터 확장 — 낙뢰(weather_code)·돌풍(wind_gusts)·시정(visibility)·구름·적설 (점수 계산엔 미사용, 러닝 골든마스터 불변)
- 활동 tagline 노출(탭 아래 + 드로어 설명), 검색창 활동별 문구
- 테스트 17개로 확대 (hike 민감도·getHikePlan·데이파트·isMountain)

### Changed
- **카테고리 순서(수요순)**: 걷기 → 애견산책 → 러닝 → 등산 → 자전거. 신규 기본 탭 = 걷기
- **오늘의 한마디 전면 심화** — 5개 활동 × 조건별 3~4개, 수요 밀착 카피 (애견은 강아지를 아끼는 진심 톤)
- 앱 메타·매니페스트·가이드가 야외활동 전반으로 확장 (드로어 가이드 7종 전면 개편: 활동별 점수 기준·등산 전 체크·애견산책 가이드 등)
- 5개 탭 균등 정렬 — iPhone SE(360)~와이드 PC까지 한 줄 유지
- 데이파트 섹션 제목 동적 ("새벽·아침·낮·저녁 베스트")

### Removed
- 미사용 AI 팁 API(app/api/tip)

### Invariant
- 러닝 골든마스터 계속 통과 — 러닝 점수·톤·멘트 불변

## [0.3.1] - 2026-07-05
PC 화면 최적화 + 활동 탭 정렬.

### Changed
- **데스크탑 2단 대시보드** — 넓은 화면(≥1081px)에서 좌(히어로+추천시간)/우(일출일몰·시간대·복장·한마디·지표) 2열로 공간 활용. 모바일은 `display:contents`로 기존 1단 그대로 유지

- **활동 탭 4개를 한 줄에 균등 정렬**(flex:1) — 모바일·태블릿·데스크탑 모두 스크롤 없이 딱 맞음. 탭 순서 기준 확정(브랜드 앵커 러닝 + 사용자 범용성: 러닝→걷기→애견산책→자전거)
- 카카오 위치 검색 연동(환경변수 KAKAO_REST_API_KEY) 반영

## [0.3.0] - 2026-07-05
멀티 활동 완성도 개선. 출퇴근을 자전거로 교체하고, 러닝 표현 누수를 전면 정리하고, 색상 위계를 재설계.

### Added
- **자전거 카테고리** 🚴 — 출퇴근을 대체. 바람 가중 최상(0.22) + 강풍 위험캡(11m/s→38점), 방풍·헬멧·아이웨어·라이트 복장, 라이딩 전용 문구, 강풍 안내 배너
- **애견 전용 카드** — 산책 신호등(가능/주의/자제) + 추천 산책 길이 + 발바닥 상태 + 챙길 것 체크리스트
- 활동별 UI 문구 중앙관리(`ActivityProfile.terms`) — 우중런/복장/알림/차단 태그 등

### Changed
- **텍스트 전면 활동화** — "우중런"→활동별(우중 걷기/비 산책/빗속 라이딩), "러닝 복장/알림/기록" 등 공용 문구가 선택 활동을 반영
- **색상 위계 재설계** — 1등 추천 카드의 앰버 색면 제거 → 흰 바탕 + 골드 좌측 액센트바(일출 카드와 색 충돌 해소), 일출 웜/일몰 쿨 대비 강화
- **카테고리 선택 상단 탭으로 통일** — 데스크탑 좌측 세로 레일 제거, 상단 가로 탭 중앙정렬
- 활동 기록·알림 문구를 활동 무관 중립 표현으로

### Removed
- 출퇴근(commute) 카테고리 및 우산 안내

### Invariant
- 러닝 골든마스터 계속 통과 — 러닝 점수·톤·멘트 불변. windCap run={14,40}로 기존과 동일.

## [0.2.0] - 2026-07-05
러닝 전용 → 멀티 활동 앱으로 확장. 러닝은 기존 그대로, 걷기·애견산책·출퇴근 추가.

### Added
- **활동 카테고리 4종** — 🏃러닝 / 🚶걷기 / 🐕애견산책 / ☂️출퇴근. 선택하면 점수·추천시간·문구·복장이 그 활동 기준으로 전부 재계산
- 활동 선택 UI — 데스크탑 좌측 레일, 모바일 상단 가로 탭, 드로어 메뉴. 선택은 저장됨
- **애견산책 발바닥 경고** — 한낮·더위·자외선 조합 시 지면 열기/화상 위험 배너 (안심/주의/위험 3단계)
- **출퇴근 우산 안내** — 앞으로 12시간 비 소식 기반 "우산 없이 OK / 접이식 정도 / 우산 챙기세요 / 지금 비" 안내
- 활동별 전용 카피 전면 — 헤드라인·서브라인·오늘의 한마디·코치 브리핑·지표 설명·복장 추천
- 활동별 점수 프로필 — 걷기는 더위·습도 관대 + 미세·자외선 강조, 애견은 더위 엄격, 출퇴근은 강수 최우선
- vitest 테스트 환경 + **러닝 골든마스터 테스트** (러닝 점수·멘트 불변 잠금)

### Changed
- 예보 API가 원본 데이터를 반환하고 점수화는 클라이언트에서 — 활동 전환이 재요청 없이 즉시 반영
- "나의 러닝 기록" → "나의 활동 기록"
- 공유 문구가 선택된 활동을 반영

### Invariant
- 러닝 카테고리는 점수·문구·화면 모두 v0.1.0과 동일 (골든마스터 12케이스 + 브라우저 검증)

## [0.1.0] - 2026-07-05
첫 버전 고정. 지금까지 만든 기능을 하나의 baseline으로 확정.

### Added
- 러닝 컨디션 점수 (0~100) — 날씨 + 대기질 기반, 강수/체감온도/바람 하드캡
- 시간대별 최적 러닝 시간 추천 (1·2·3순위, 새벽/야간 제외)
- 6개 지표 상세 시트 + 인터랙티브 차트 (체감온도·강수·미세먼지·자외선·바람·습도)
- 시간대별 복장 추천 (날씨 변화 반영, 자외선 시 선글라스 등)
- 러닝 기록·스트릭·캘린더 + 주간 목표 설정
- 위치 검색 (카카오 Local API, Nominatim 폴백) + 즐겨찾기/최근
- 어제·오늘·내일 연속 타임라인 (Open-Meteo, past_days=1 / forecast_days=2)
- PWA 설치 + 알림 (Notification Triggers, 미지원 시 setTimeout 폴백)
- 브라이트 에너지 디자인 (코발트 + 라임), Pretendard/Black Han Sans

### Tech
- Next.js 16 (Turbopack) + React 19 + TypeScript
- 코드 약 6,100줄, `app/page.tsx`에 UI 집중
