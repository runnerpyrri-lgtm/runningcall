## [0.8.0] - 2026-07-06
"출발선(Start Signal)" 리디자인 — 아스팔트 네이비 다크 테마 + 신호등 시그널 시스템.

### Added
- **활동 픽토그램 5종**(`lib/pictograms.tsx`) — 러닝·걷기·애견산책·등산·자전거를 신호등 스타일 SVG로. 활동 탭·PC 레일·드로어·히어로 다이얼에서 사용, 선택 시 시그널 그린 점등
- **신호 다이얼 히어로** — 픽토그램 + 대형 점수 + 등급칩("좋음 · GO"/"보통"/"주의"/"나쁨")이 한 몸. 링이 점수만큼 시그널 색으로 차오르고 글로우가 숨쉬듯 박동
- 시그널 컬러 시스템 — 점수 구간(72/55/38)에 따라 다이얼·픽토그램·등급칩·글로우가 초록/앰버/오렌지/레드로 일괄 전환

### Changed
- **다크 "출발선" 테마** — 전 화면을 아스팔트 네이비(#0a0f18) 기반으로 전환. 디자인 토큰 플립 + 하드코딩 라이트 색 정리(서피스·틴트·보더·텍스트)
- 활동 탭·드로어 선택 상태를 브랜드 블루에서 시그널 그린 틴트로 변경

### Fixed
- 도전과제 시리즈 버튼의 UA 기본 흰색 outset 보더 노출(다크 배경에서 드러난 잠복 버그) — 전역 버튼 보더 리셋

### Invariant
- 러닝 점수 가중치·상수 불변(골든마스터). 테스트 47개 통과, 프로덕션 빌드 정상

## [0.7.0] - 2026-07-05
뱃지를 3000개 이상으로 대폭 확장 + 도전과제 컬렉션 화면.

### Added
- **뱃지 3,070개 — 시리즈 생성기** — 손으로 나열하지 않고 지표별 시리즈에 촘촘한 단계 사다리를 깔아 생성
  - **초반 밀도로 포기 방지**: 대부분 시리즈가 1·2·3·4·5…로 시작 → 처음엔 운동할 때마다 뱃지가 우수수(테스트에서 러닝 25km 한 번에 68개 획득 확인)
  - 31개 시리즈: 총 운동·활동별 횟수 / 러닝·자전거·합계 거리 / 활동별·합계 시간 / 연속일·한 달 최다일 / 산 정복 / 한 방 최고 거리·최장 시간 / 일기·기분·최고의 날 / 다양성
- **도전과제 컬렉션 화면** — 3000개를 평면 나열하지 않고 시리즈 카드로
  - 상단 진행 링 + `달성 N / 3,070`
  - 시리즈 카드: 이모지·이름·`Lv N`·다음 목표·진행바. 탭하면 **아코디언으로 그 시리즈 전체 단계** 펼침(달성=컬러, 다음 목표=강조)
- 축하 팝업이 다수 동시 달성 시 상위 4개 + `외 N개 달성!`로 요약

### Changed
- 달성 통계에 활동별 시간·총 거리·총 시간·한 번 최장 시간 추가(일지 데이터 기반, 소급)

### Invariant
- 러닝 골든마스터 통과. 테스트 47개. 데이터 스키마 불변.
