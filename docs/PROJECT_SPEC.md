# 야외봄 (OutBom) — 프로젝트 명세

## 한 줄 정의
"지금 뛰어도 돼? 오늘 언제가 최적?"에 **3초 안에** 답하는 러닝 컨디션 습관 서비스.

## 대상 사용자
- 야외 러닝을 하는 사람 중, 날씨·미세먼지 때문에 "지금 나갈까 말까" 매번 고민하는 사람
- 러닝 습관을 만들고 싶어 기록/스트릭에 동기부여 받는 사람

## 핵심 가치
1. **판단을 대신해준다** — 여러 날씨 지표를 러닝 관점 점수 하나로 요약
2. **최적 시간을 짚어준다** — 오늘 남은 시간 중 1·2·3순위 러닝 창(window)
3. **습관을 쌓게 한다** — 기록·스트릭·주간 목표

## 현재 기능 (v0.1.0)
- 러닝 점수(0~100): 기온/체감/강수/바람/미세먼지/자외선/습도 종합, 위험요소 하드캡
- 시간대별 추천(1·2·3순위), 복장 추천(시간대·날씨변화 반영)
- 6개 지표 상세 + 인터랙티브 차트, 어제~내일 타임라인
- 러닝 기록/스트릭/캘린더, 위치 검색·즐겨찾기, PWA 설치·알림

## 기술 스택
| 영역 | 현재 (v0.1.0) | 향후 |
|---|---|---|
| 프론트/웹 | Next.js 16 + React 19 + TS | 유지 |
| 데이터 | Open-Meteo(날씨/대기질), Kakao Local(위치) | 유지 |
| 배포 | (로컬만) → **Vercel** | 유지 |
| 회원/DB | 없음 | **Supabase** (Auth + DB) |
| 앱 | PWA | TWA/Capacitor 감싸기 또는 Expo (미결정) |

## 최종 목표 흐름
웹 상시배포 → 도메인 → 회원가입/로그인 → iOS·Android 출시 → (AdSense 등) 수익화.
버전이 쌓이는 방식으로 운영 (생각나는 대로 X, 기록 기반 O).

## 비목표 (넣지 않기로)
- 달/월령, 측정소 지도 등 러닝과 무관한 정보 과잉
- 요청되지 않은 과도한 추상화/설정 (YAGNI)
