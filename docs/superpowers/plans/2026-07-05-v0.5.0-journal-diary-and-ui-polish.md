# v0.5.0 운동 일지 다이어리 + UI 폴리시 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans. Steps use checkbox (`- [ ]`) syntax.

**Goal:** 운동 일지를 활동별 가벼운 기록 + 기분 + 자유 일기의 다이어리 앱으로 심화하고, 캘린더 짤림·애견 두 줄·준비 밋밋함·드로어 장황함·이모지 임팩트를 함께 해결한다.

**Architecture:** journal.ts를 v2(활동=객체+기분+메모)로 확장하고 v1 무손실 마이그레이션. JournalView는 캘린더↔편집기 두 화면 + draft/저장·취소. 나머지는 PrepView·드로어·CSS 폴리시.

**Tech Stack:** Next.js 16 / React 19 / TypeScript / vitest / localStorage.

## Global Constraints
- 러닝 골든마스터 통과 유지.
- 기존 `running-alarm:journal:v1` 무손실 마이그레이션 → `running-alarm:journal:v2`.
- 등산 4구간 유지.
- 신규 파일 첫 줄 한국어 역할 주석.
- 각 태스크: `pnpm test && pnpm typecheck && pnpm build` 통과 후 커밋.
