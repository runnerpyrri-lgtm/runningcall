# v0.7 뱃지 3000 시리즈 + UI 폴리시 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans. Steps use checkbox (`- [ ]`) 문법.

**Goal:** 뱃지를 시리즈 생성기로 3000개 이상 만들어 초반 밀도로 포기를 막고(v0.7.0), 밝은 정체성 유지한 채 UI 완성도를 정교화한다(v0.7.1).

**Architecture:** achievements.ts를 시리즈 생성기로 확장(통계 추가·SERIES·evaluateSeries). AchievementsView를 컬렉션(시리즈 카드+아코디언)으로 재작성, 축하 요약. 이후 디자인 토큰·마이크로 인터랙션 폴리시.

**Tech Stack:** Next.js 16 / React 19 / TypeScript / vitest / localStorage.

## Global Constraints
- 러닝 골든마스터 통과, 기존 데이터 스키마 불변·소급.
- 브랜드색·레이아웃 구조 유지.
- 각 릴리스: `pnpm test && pnpm typecheck && pnpm build` + 브라우저(360/1440) 후 태그·배포·200.
