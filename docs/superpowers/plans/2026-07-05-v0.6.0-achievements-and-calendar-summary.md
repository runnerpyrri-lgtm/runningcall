# v0.6.0 도전과제/뱃지 + 성과 요약 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans. Steps use checkbox (`- [ ]`) 문법.

**Goal:** 캘린더에서 날짜 성과를 바로 확인하고 기록 있는 날은 빨간 링으로 표시하며, 일지 밑 도전과제 카테고리에서 36개 뱃지를 소급 달성·수집하고 새 달성마다 축하 팝업을 띄운다.

**Architecture:** 순수 함수 엔진(lib/achievements.ts)이 journal+activity-log에서 통계·뱃지를 계산. JournalView에 요약 패널·빨간 링, 신규 AchievementsView·CelebrationModal, Home 배선.

**Tech Stack:** Next.js 16 / React 19 / TypeScript / vitest / localStorage.

## Global Constraints
- 러닝 골든마스터 통과 유지.
- 기존 데이터(journal v2·activity-log) 읽기만, 스키마 불변.
- 뱃지 과거 소급, 첫 실행은 조용히 baseline seen 처리.
- 신규 파일 첫 줄 한국어 주석.
- 각 태스크: `pnpm test && pnpm typecheck && pnpm build` 통과 후 커밋.
