  개별 수정(전수 리라이트 금지 — 서지컬 원칙).

  전역 `button { border: 0 }` 리셋으로 해결 — 모든 스타일드 버튼은 자기
  보더를 명시하므로 안전. 새 버튼 만들 때 보더 필요하면 직접 선언할 것.

  class 스타일이 안 먹음(검증 페이지 한정 이슈, React 인라인은 무관).
